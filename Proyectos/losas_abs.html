Descripción

El objetivo es reciclar el plástico ABS para realizar elementos para la construcción desde un entorno casero, con herramientas caseras y con la menor contaminación posible.

 

Plásticos

Se está experimentando con plástico ABS llamado plástico de ingeniería de los del Tipo 7. Es un plástico usado en chasis de electrodomésticos.

 

Proceso

 

Colectar

Se colectan en una planta de recolección. Este proceso se realiza por un tercero por lo cual no tengo conocimiento de cómo se hace en detalle.

 

Seleccionar

Se selecciona en una planta de recolección. Este proceso se realiza por un tercero por lo cual no tengo conocimiento de cómo se hace en detalle

 

Triturar

Se tritura en una planta con máquinas semindustriales o industriales. Las partículas serán del tamaño alrededor de 5 mm.

 

Moldear

Se utilizan moldes metálicos hechos con las chatarra recuperada de los mismos equipos electrodoméstico de donde viene el plástico. Para cortar se emplearon tijeras hojalateras y pulidora.

Al molde se le unta aceite, cualquiera que no sea el de freír, logrando una película fina y luego se vierten las partículas de plásticos y se acomodan. Se hace un capa fina buscando que sea uniforme. Se le rocía acetona y  se tapa para que los vapores actúen diluyendo el plástico. Se estima el tiempo a que se disuelva alrededor de unos 30 min y luego se va a vertir más plástico y repetir todo el proceso.

 

Hornear

Se emplea un horno eléctrico casero marca Black & Decker. Se introduce el molde en el horno con tapa presionadas con prensas de tipo C . Se deja hornear a 235 grados por 20 minutos.

 

Protección

Evitar las quemaduras:

Usar guantes para temperatura los de soldar son ideales.

Desmoldar con cuidado.

Evitar respirar los gases tóxicos:

Usar máscara para gases

Realizar la cocción al aire libre.

Esperar a que se enfríe el horno a temperatura ambiental.

 

Producto

 

Losas de piso

Se aplica un acabado al producto con cuchilla y lija para madera. Se puede optar por baño de vapores de acetona, (como el que se le aplican a las piezas de impresión 3d).

 

Pruebas

Se realizan probetas de losas de pisos y perfiles cuadrados. Para la investigación se realizaron cuatro pruebas. El costo de toda la investigación fue de aproximadamente de 1000 pesos mexicanos (sin incluir la recolección ni trituración).
